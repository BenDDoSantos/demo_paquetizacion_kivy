# Demostración de paquetización

Aplicación de Kivy paquetizada con Buildozer
La aplicación contiene un texto simple, además de tener un pequeño botón interactivo
El repositorio contiene:

```
main.py - Archivo donde contiene el codigo de la aplicación
logo.png - Imagen usada como icono de la aplicación
Hola.apk - Aplicación paquetizada y disponible para su descarga
Informe_BDS.pdf - Archivo donde se comenta como se hizo el procedimiento de paquetización
```
